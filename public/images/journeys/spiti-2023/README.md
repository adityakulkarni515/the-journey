## Image Recommendations

1. **Format**: JPG for photos, PNG for graphics
2. **Hero image**: At least 1920px wide, landscape orientation
3. **Moments**: Square or 4:3 ratio works best
4. **File size**: Compress images (use tinypng.com) for faster loading
